<h2>My projects</h2>

<h3>1. Anime Downloader</h3>
I made this because it was often frustrating to download my anime with all the annoying ads, dealing with all the pop-ups as well as handling non-batch downloads. This repository focuses on a clean CLI that scrapes a list of pages for links, identifies the ones required by the user and downloads them using JDownloader all whilst naming in the required convention so that even Media Servers such as Plex will be able to recognise and operate.

<h3>2. Zoom Joiner</h3>
This repository is also constructed in Python. It requires users to input information regarding their meetings as well as the respective links. It then automatically runs everyday and connects users to these meetings without going through the hassle of checking for times and identifying the necessary links.
